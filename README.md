# CCNA Job Tracker Bot

A Python automation project that searches for entry-level networking, NOC, and CCNA-oriented jobs near San Antonio, Texas.

## Project Goals

- Search for entry-level networking roles
- Find CCNA, NOC, help desk, and network technician job postings
- Save matching jobs to a CSV file
- Track jobs I have reviewed or applied to
- Eventually run automatically every day

## Why I Built This

I am studying for the CCNA and building hands-on projects to prepare for an entry-level networking or NOC technician role.

This project helps me learn Python while also supporting my job search.
